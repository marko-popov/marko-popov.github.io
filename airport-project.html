Airport Gate Management System

Java Spring Boot & PostgreSQL Solution

Project Overview

This project was designed to solve the logistical challenge of allocating airport gates to incoming flights efficiently. The system ensures that no two flights are assigned to the same gate at the same time, while providing real-time scheduling updates.

Architecture

The system follows a monolithic RESTful architecture with a focus on clean code and separation of concerns.

  • Backend: Java 17, Spring Boot, Spring Data JPA.
  • Security: JWT (JSON Web Tokens) for role-based access control (Admin/Staff).
  • Database: PostgreSQL for persistent storage.
  • Tools: Docker for containerization, Swagger for API documentation.
