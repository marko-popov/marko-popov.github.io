Airport Gate Management System

Java Spring Boot PostgreSQL JWT Auth

Project Overview

This project was designed to solve the logistical challenge of allocating airport gates to incoming flights efficiently. The system ensures that no two flights are assigned to the same gate at the same time, while providing real-time scheduling updates.

Key Focus: Resource Optimization & Security

Architecture & Tech Stack

Backend Layer

Built with Java 17 and Spring Boot, utilizing Spring Data JPA and Hibernate for ORM-based persistence.

Security Model

Implemented Spring Security with Stateless JWT authentication to ensure secure API access for different user roles.

Infrastructure

Containerized using Docker and Docker Compose. PostgreSQL is used for reliable and persistent data storage.

Workflow & Process

1

Flight Request

User submits flight number via secured REST endpoint.

2

Validation

Service validates arrival time and gate operating window.

3

Availability Filtering

Repository filters gates by time and occupancy constraints.

4

Transactional Assignment

Gate assignment executed atomically within transactional boundary.

5

Secure Response

Assigned gate returned as JSON response with role validation.
